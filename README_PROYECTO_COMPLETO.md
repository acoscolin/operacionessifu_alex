## 🐛 **SOLUCIÓN DE PROBLEMAS COMUNES**

### **No aparecen los módulos en Smart Hub**
1. Abre la consola (F12)
2. Busca errores en rojo
3. Verifica que todos los archivos JS estén cargados
4. Recarga la página (Ctrl+F5)

### **No se puede instalar la app**
1. Verifica que los iconos existan (`icon-192.png`, `icon-512.png`)
2. Comprueba que `manifest.json` esté accesible
3. Asegúrate de estar en HTTPS o localhost
4. Revisa la consola del navegador

### **El chat no funciona**
1. Verifica que localStorage esté habilitado
2. Comprueba que `internal_chat.js` esté cargado
3. Abre la consola y busca errores
4. Recarga la página

### **Modo offline no funciona**
1. Verifica que el Service Worker esté registrado
2. Abre DevTools → Application → Service Workers
3. Comprueba que los archivos estén cacheados
4. Recarga la página
